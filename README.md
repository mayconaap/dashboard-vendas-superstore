## 📋 **Descrição do Projeto**

Este é o **segundo projeto** do meu portfólio de análise de dados, desenvolvido durante o curso **Santander + DIO**. O objetivo é criar um **dashboard interativo no Excel** a partir da base de dados **Superstore**, uma das bases mais famosas para análises de vendas.

> 🎯 **Propósito:** Consolidar conhecimentos em Excel, tabelas dinâmicas, gráficos e segmentações de dados.

## ✨ **Funcionalidades do Dashboard**

### **KPIs Principais**
- ✅ **Faturamento total** do período
- ✅ **Lucro total** das vendas
- ✅ **Quantidade de pedidos** realizados
- ✅ **Ticket médio** por venda

### **Análises Gráficas**
- 📊 **Vendas por categoria** (móveis, tecnologia, material de escritório)
- 🌎 **Vendas por região** (análise geográfica)
- 📈 **Evolução mensal** das vendas
- 🏆 **Top 10 produtos** mais vendidos
- 👥 **Análise por segmento** de cliente

### **Interatividade**
- 🔍 **Segmentações (slicers)** para filtrar por:
  - Ano
  - Região
  - Categoria
  - Segmento de cliente
